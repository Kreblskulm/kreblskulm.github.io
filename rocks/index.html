Stuff that rocks!

Here is a collection of things that I personally think "rock"!

Linux distributions:

  • Fedora - Fedora is my favourite Linux distribution. Fedora is a release-point Linux distribution which integrates newer technologies such as the latest desktop environment, kernel release, wayland and other technologies. At the moment Fedora Workstation is my distribution of choice.
    • Fedora Silverblue - Fedora Silverblue provides all of the features of Fedora Workstation as an immutable desktop operating system.
  • Pop!_OS - Pop!_OS enhances the Linux experience with their own custom desktop workflow, updated kernels, out of the box Nvidia support and many other great optimisations to the Linux desktop.
  • Debian - Debian is a very stable Linux distribution with a large focus on free software. The project has thousands of contributors and a large community around it. Many other Linux distributions are based on Debian.
  • Arch Linux - Arch Linux is a great, rolling-release DIY distribution which provides great community-driven documentation.

Desktop environments and Window Managers:

  • GNOME - A beautiful, innovative GTK desktop environment pushing bleeding-edge technologies in the Linux desktop such as Wayland and XDG portals.
  • KDE Plasma - A beautiful, customizable Qt desktop environment pushing bleeding-edge technologies in the Linux desktop such as Wayland and XDG portals.
  • XFCE - A lightweight desktop environment for UNIX-like operating systems. XFCE aims to be fast and low on system resources, whilst still remaining user-friendly.
  • SwayWM - Sway is a tiling wayland compositor and wayland replacement for i3wm. It supports most of i3's features and works well with current i3 configurations.

Package managers:

  • Flatpak - Universal package manager providing sandboxing (security), containerization and desktop integration.
  • rpm-ostree - The package manager used in Fedora Silverblue. Provides support for rpm packages whilst retaining the immutability features found in ostree.

Web Browsers:

  • Firefox - A very popular free and open-source web browser developed by Mozilla with many great features.
  • GNOME Web - An open source web browser with a built in ad-blocker developed by the GNOME foundation.

Chat services:

  • Matrix/Element - A Secure, decentralized, federated, and open source chat platform with modern features. Can be used as a Discord alternative to some extent.

Websites, blogs and write-ups:

  • Response to Flatkill.org - A well-written blog post responding to and debunking claims made by Flatkill.
  • OMG! Ubuntu! - A brrilliant blog page which, while focussing on Ubuntu, also covers other important news in the Linux-sphere!
  • Fedora Magazine - The official blog page for all things Fedora! Ran by the Fedora community, for the Fedora community!

Games and Game services:

  • Mindustry - Cross-platform, free and open source hybrid tower defense and factory game.
  • Steam - A (proprietary) gaming platform and launcher which is currently peddling support for gaming on Linux.
