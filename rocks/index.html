Stuff that rocks

Here are some awesome things that I personally think "rock":

Linux distributions:

  • Fedora - Probably one of my favourite Linux distributions. Fedora is a release-point Linux distribution which integrates newer technologies such as the latest desktop environment, kernel release, wayland and other technologies.
  • Arch Linux - A rolling release, DIY distribution that has great documentation.
  • Ubuntu - A brilliant user-friendly "just-works" distribution with a lot of support online.
  • Pop!_OS - Ubuntu-based distribution with a customized desktop experience. Great NVIDIA and gaming support.
  • Debian - A Linux distribution which focuses on stability.

Desktop environments and window managers:

  • GNOME - A beautiful, innovative GTK desktop environment pushing bleeding-edge technologies in the Linux desktop such as Wayland and XDG portals.
  • KDE Plasma - A beautiful, customizable Qt desktop environment pushing bleeding-edge technologies in the Linux desktop such as Wayland and XDG portals.
  • XFCE - A lightweight desktop environment for UNIX-like operating systems. XFCE aims to be fast and low on system resources, whilst still remaining user-friendly.
  • i3wm - A highly configurable, easy-to-configure window manager written from scratch. i3wm uses newer technologies such as xcb rather than Xlib.
  • Sway - Sway is a tiling wayland compositor and wayland replacement for i3wm. It supports most of i3's features and works well with current i3 configurations.

Here are some awesome things that I personally think "rock":

  • Flatpak - Universal package manager providing sandboxing (security), containerization and desktop integration.

Text editors:

  • Code - OSS - An open source repository for Visual Studio Code. Code has many different themes and extensions which make programming easier.
  • GNU nano - A simple, open source, command-line text editor.

Web browsers:

  • Firefox - Very popular free and open-source web browser developed by Mozilla with many great features.
  • GNOME Web (Epiphany) - An open source web browser with a built in ad-blocker developed by the GNOME foundation.

Chat services:

  • Matrix/Element - A Secure, decentralized, federated, and open source chat platform with modern features. Can be a Discord alternative to some extent.
  • SchildiChat - A Matrix client based on Element with an IRC-Style User interface.
  • Session - A secure, decentralised, federated and anonymous chat platform based on Signal.

Websites, blogs and write-ups:

  • Response to Flatkill.org - A well-written blog post responding to and debunking claims made by Flatkill.

Others:

  • Cryptpad - An open source, end-to-end encrypted collaboration suite and alternative to the google suite.
  • LibreOffice - A feature rich, open source office platform which is a great, compatible alternative to Microsoft Office.
